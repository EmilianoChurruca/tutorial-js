Las funciones const, let y var



Para tener en cuenta:

  • En el ejemplo saludar se muestra el string a través de la consola
  • Cuando se escribe el nombre de la función y se colocan los paréntesis, se llama a la misma.
  • También se puede guardar en una variable y para invocarla, se escribe el nombre de la variable. Así está el ejemplo del alumno con su respectivo argumento entre los paréntesis.

Otra forma de escribir las funciones es con lo que se denomina arrow function, una forma más simplificada.


Se pasa el nombre de la función por detrás del paréntesis y se coloca el símbolo =>

En el ejemplo se ve que no se escribe la palabra function y se escribe sólo el parámetro, algunos ejemplos:

  • Sin parámetros: let saludar = () => {return "hola"}
  • Con un parámetro: let saludar = nombre => {return `hola ${nombre}`} Nota: se pueden omitir los paréntesis
  • Con más de un parámetro: let saludar = (nombre1, nombre2) => {return `hola ${nombre1} y ${nombre2}`}

Una forma más simplificada de arrow function quitando las llaves, a continuación los ejemplos.
