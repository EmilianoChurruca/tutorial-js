Import-Export

En JavaScripts se puede exportar de un archivo js a otro, para ello debemos hacer:

  • Un archivo que exporte: Variables, default, objetos, funciones, clases y clases con métodos. A continuación un ejemplo de lo mencionado
  • Y otro que importe y utilice lo importado para lo que sea: Aclaración: el import debe estar arriba.

Luego, con el import, tenemos disponible lo exportado y a partir de ahí se puede utiliiza como si estuviera declarado en el mismo archivo
