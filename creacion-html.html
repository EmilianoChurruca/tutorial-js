Crear-html

En esta parte vamos a ver cómo se genera código html a través de JavaScript, de manera tradicional se hay que acordarse de la regla "contenedor/contenido", primero genero un array con colores

Defino un contenedor, lo apunto según el id que le puse en el documento html y defino el contenido con una cadeba vacía

Luego se crea la cadena, iterando sobre el array. Hay varias maneras, (while - for). Por ejemplo el forEach itera sobre los elementos del array y la variable "color" va tomando el valor de los elementos. Luego se "renderiza" con innerHTML
