Definición de variables

En JavaScript existen tres tipos de variables const, let y var



Para tener en cuenta:

  • const: No se puede modificar, el valor se mantiene constante
  • let: es una variable que tiene alcance sólo donde está definida
  • var: es una variable global, si se modifica, la misma cambia no importa en dónde se modificó
